Simulador de Automatización Industrial + Material Educativo Abierto

Este repositorio contiene un entorno de simulación didáctico para el aprendizaje de automatización industrial, programación de PLC y lógica Ladder, acompañado por cuadernillos teóricos y ejercicios progresivos.

El proyecto fue desarrollado con fines educativos y está pensado para:

- estudiantes de electrónica y automatización
- docentes técnicos
- autodidactas
- instituciones educativas
- makers e interesados en control industrial

Incluye:
- simulador interactivo
- ejemplos de programas
- ejercicios guiados
- material teórico estructurado
- actividades prácticas

Filosofía del proyecto:
Este material se comparte bajo un enfoque de conocimiento abierto, promoviendo el aprendizaje libre, la mejora colaborativa y la difusión del saber técnico.
